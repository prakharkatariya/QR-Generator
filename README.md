# QR Code Generator

A simple, modern, and premium QR Code Generator web application.

## Features

- **Instant Generation**: Generate QR codes instantly from any URL or text.
- **Premium Design**: Glassmorphism UI with smooth animations.
- **Downloadable**: Download your generated QR code as a PNG image.
- **Responsive**: Works beautifully on all devices.

## How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/prakharkatariya/QR-Generator.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd QR-Generator
    ```
3.  **Open `index.html`:**
    Simply double-click `index.html` to open it in your default web browser. No server installation is required!

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, Animations)
- JavaScript (Vanilla)
- [qrcode.js](https://davidshimjs.github.io/qrcodejs/) library
